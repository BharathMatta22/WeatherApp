# WeatherApp
This project is a simple weather application built using React.js for the frontend and .NET for the backend.
The frontend displays the weather information and forecast in a user-friendly interface.
This project is a simple weather application built using React.js for the frontend and .NET for the backend.
It allows users to check the current weather and forecast for a specific city.
The backend retrieves weather data from the OpenWeatherMap API and stores it in a SQL Datbase.
The frontend displays the weather information and forecast in a user-friendly interface with the previous five recent searches.
We used an external api to fetch them.
